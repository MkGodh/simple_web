О проекте

Simple Web — это учебный проект на C++, реализующий асинхронное TCP-взаимодействие между сервером и клиентом с использованием библиотеки Boost.Asio.

Проект демонстрирует:

работу с TCP-сокетами
асинхронные операции (async_accept, async_write, async_receive)
архитектуру клиент–сервер
управление соединениями через shared_ptr
🛠️ Технологии
C++17
Boost.Asio
CMake
