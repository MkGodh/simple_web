Simple Web (TCP Server/Client on Boost.Asio)

##  О проекте

**Simple Web** — это учебный проект на C++, реализующий асинхронное TCP-взаимодействие между сервером и клиентом с использованием библиотеки Boost.Asio.

Проект демонстрирует:
- работу с TCP-сокетами  
- асинхронные операции (`async_accept`, `async_write`, `async_receive`)  
- архитектуру клиент–сервер  
- управление соединениями через `shared_ptr`  

---

## 🛠️ Технологии

- C++17  
- Boost.Asio  
- CMake  

---

##  Архитектура

###  Сервер

- **TCPserver**
  - Принимает входящие подключения
  - Использует `boost::asio::async_accept`
  - Хранит список активных соединений

- **TCPConnection**
  - Представляет одно клиентское соединение
  - Отправляет сообщение клиенту
  - Обрабатывает отключение клиента

###  Клиент

- Подключается к `127.0.0.1:8080`
- Читает данные от сервера в бесконечном цикле
- Выводит полученные сообщения в консоль

---

##  Сборка проекта

- git clone https://github.com/MkGodh/simple_web.git
- cd simple_web

- mkdir build
- cd build

- cmake ..
- cmake --build .

# Запуск

## Запуск сервера
```bash
./gdhServer
```

Сервер начнёт слушать порт **8080**

## Запуск клиента
```bash
./gdhClient
```

Если всё успешно:


```text
Connected!
```
# Как это работает

## Сервер
- Запускается и ждёт подключения
- Клиент подключается
- Создаёт `TCPConnection`
- Отправляет сообщение (`async_write`)

## Клиент
- Получает данные (`read_some`)
- Выводит их в консоль

#  Особенности реализации

- Используется `shared_from_this()` для безопасной работы с async callbacks  
- Асинхронная модель без блокировок  
- Сервер поддерживает несколько клиентов  
- Используется `io_context` как ядро событий  

# ⚠️ Ограничения

- Сервер не обрабатывает входящие сообщения от клиента  
- Нет протокола передачи данных  
- Нет многопоточности  
- Соединения не удаляются из контейнера  

#  Возможные улучшения

- Реализовать чат между клиентами  
- Добавить thread pool  
- Ввести протокол (например: длина + сообщение)  
- Добавить логирование  
- Реализовать корректное завершение соединений
